# Sathish Kumar Portfolio

A modern, responsive developer portfolio built with **React**, **TypeScript**, and **Vite**.

## 🚀 About

This portfolio showcases my experience as a **Full Stack Developer**, highlighting my projects, technical skills, professional experience, and education.

The application is built using modern React architecture with reusable components and TypeScript.

---

## 🛠️ Tech Stack

- React
- TypeScript
- Vite
- React Router
- CSS3
- React Icons
- Framer Motion *(coming soon)*

---

## ✨ Features

- Responsive Design
- Smooth Scrolling Navigation
- Mobile Navigation Menu
- Hero Section
- Projects Showcase
- Skills
- Experience Timeline
- Contact Section
- Resume Download
